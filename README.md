# Cinemas

Скрипт для поиска фильмов с высоким рейтингом в прокате.
---------------------------------------------------------------------------

## Запуск:

    python cinemas.py

## Описание:

Скрипт вернёт в консоль 10 фильмов с самым высоким рейтингом, которые идут в кинотеатрах на данный момент.
